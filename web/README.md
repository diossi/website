# FashionStore — E-Commerce MVP Platform

Веб-приложение (MVP) интернет-магазина одежды с полноценным бэкендом, динамическим управлением каталогом, корзиной покупок и панелью администратора (складом).

## 🚀 Технологический стек

* **Core:** Python 3.10+, Flask
* **ORM & Database:** SQLAlchemy, SQLite (локально), полностью готов к деплою на PostgreSQL (интегрирован драйвер `psycopg2-binary`)
* **Authentication:** Flask-Login (сессионная авторизация)
* **Forms & Validation:** Flask-WTF, WTForms (валидация данных на бэкенде)
* **Frontend:** Jinja2 (серверный рендеринг шаблонов), HTML5, CSS3, Flash-уведомления

## 🌟 Основной функционал

### 🔒 Безопасность и ролевая модель
* Полноценная регистрация и авторизация пользователей с валидацией email и совпадения паролей.
* Безопасное изменение текущего пароля из личного кабинета с проверкой хэша.
* Разделение прав доступа (RBAC): обычные пользователи имеют доступ к каталогу и корзине, администраторы (`is_admin`) — к управлению складскими остатками.

### 🛒 Бизнес-логика e-commerce
* **Каталог:** Динамическая фильтрация товаров по категориям и вывод похожих позиций на странице товара.
* **Транзакционная корзина:** Реализована логика добавления/удаления товаров (`WantBuyItem`). Система автоматически уменьшает доступное количество товара на складе (`item.count`) при добавлении в корзину и возвращает его обратно при удалении.
* **Оформление заказа:** Эндпоинт `/buyitems` симулирует покупку, очищает корзину пользователя и обновляет внутреннюю метрику популярности товаров (`count_buy`).

### 📦 Панель администратора (Склад)
* Изолированный эндпоинт `/storage` с проверкой прав доступа.
* Управление номенклатурой: удаление товаров со склада, быстрое изменение количества (+1 / -1) в один клик через интерактивный интерфейс.

## 📂 Структура проекта

* `main.py` — Главный модуль инициализации приложения, роутинга и бизнес-логики.
* `data/` — Инициализация базы данных и ORM-модели (`User`, `Item`, `WantBuyItem`).
* `forms/` — Классы валидации веб-форм (`RegisterForm`, `LoginForm`, `ChangePassword`).
* `templates/` — Jinja2 шаблоны страниц (каталог, корзина, профиль, админка).
* `db/` — Локальное файловое хранилище СУБД.

## 🛠 Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com
   cd FashionStore
   ```

2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Запустите сервер для разработки:
   ```bash
   python main.py
   ```
   Приложение будет доступно по адресу: `http://127.0.0`
